SpellReplay is a classic era/anniverssary/mop AddOn that was originally made for Sbkzor in order to backport a retail addon widely used by streamers called TrufiGCD. The AddOn scrolls smoothly and speeds up when multiple spells are used at the same time, making it easier to highlight good plays.

It has been built from scratch to also include and display:

Missed spells
White hits
Spell rank
Spell damage/healing
Mana gains
Pet spells
Poisons applied (or resisted) on Shiv
... and much more.
Fork of https://www.curseforge.com/wow/addons/spellreplay-quickfix which was a 
fork of https://github.com/Oldsalt0/SpellReplay 
Big thanks to Oldsalt0 for the original work.

Changes since Wrath

Moved settings to ace, and simplified it a bit. Fixed MoP specific errors.
Settings are accessible from Interface > AddOns > SpellReplay or by simply right-clicking on the background square.
